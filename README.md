An interactive resume with a review system.

## [Live Demo](cv-viewer-claudette.vercel.app)

## Key Features

- **Interactive Review System:** Users can leave reviews . 
- **Data Persistence:** All comments are saved using `LocalStorage`, so they remain even after a page refresh.
- **Dynamic Header:** Personal information is centralized in a data configuration file.
- **Advanced Responsiveness:** A sidebar that shifts its position based on screen width.
